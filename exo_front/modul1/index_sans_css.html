Au cours de cette formation nous allons avoir 2 matières :

L'intégration

Le HTML

Le HTML sert à créer le contenue du site : elle va contenir uniquement la partie textuelle du site.

LE HTML

Le CSS va nous permettre d'ajouter la couche graphique à notre site : les couleurs, les formes, le fond, etc.

Le développement

Le Javascript

Javascript est une langage de prgrammation de script employé pour rendre une page web interactive, le comportement de lapage et son apparence pourra varier en fonction de certain paramètres

Le PHP

Le PHP est un langange de progammation libre, utilisé pour généré des pages web dynamiques, qui sont donc modifiés en fonction de certains informations (heure, nom de utlisateur, formulaire rempli par l'utilisateur)

MySQL :

MySQL est un système de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple : formulaire de création de compte) pour ensuite être ré-utilisé au moment approprié (ex: connexion à un site avec identifiant et mot de passe).
